Rounding down and squaring a floating point number

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a Java program that takes the input of a floating point number, rounds it down to the nearest integer and then prints the square of that integer. For example, if the number is 4.8, you will round it down to 4, then square it to get 16. If the number is -3.2, you will round it down to -4 and then square it to get 16.
Sample Input:
4.8

Sample Output:
16


Sample Input:
-3.2

Sample Output:
16



Memory limit: 256 MB
Time limit: 5 seconds
